# 🧮 Calculadora Python (Professional Edition)

Projeto de calculadora desenvolvido em **Python**, focado em boas práticas de engenharia de software, utilizando **Poetry** para gestão de dependências e **MkDocs** para documentação automatizada.

---

### 🚀 Tecnologias e Ferramentas
* **Linguagem:** Python 3.10+
* **Gerenciador de Pacotes:** [Poetry](https://python-poetry.org/)
* **Documentação:** [MkDocs](https://www.mkdocs.org/)
* **Testes:** Pytest

---
